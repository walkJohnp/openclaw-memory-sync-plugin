# Claude Code Project Context

## Project: OpenClaw Memory Sync Plugin

### Overview
TypeScript plugin for syncing OpenClaw Agent memory files to a sync service.

### Architecture
- Plugin → HTTP API → Sync Service (Go/Gin)
- Supports incremental sync with SHA256 hash comparison
- File watcher and scheduler for automatic sync

### Key Files
- `src/sync.ts` - SyncEngine, main sync logic
- `src/scanner.ts` - MemoryScanner, file scanning
- `src/service-client.ts` - HTTP client for sync service
- `src/scheduler.ts` - Periodic sync scheduler
- `src/types.ts` - TypeScript type definitions

### Commands
```bash
npm run build      # Build TypeScript
npm test           # Run tests
npm run sync       # Manual sync
npm run scheduler  # Start scheduler (1 min interval)
```
