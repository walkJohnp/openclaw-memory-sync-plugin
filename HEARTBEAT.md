# Memory Sync Plugin - Heartbeat Tasks

## 自动完善任务（每20分钟）

每次心跳检查执行：

1. **运行测试** - `npm test`
   - 扫描器测试
   - 配置管理测试
   - 同步引擎测试
   - 哈希工具测试
   - 日志工具测试

2. **构建项目** - `npm run build`
   - TypeScript 编译
   - 类型检查

3. **覆盖率检查** - `npm run test:coverage`
   - 目标覆盖率: >80%
   - 生成覆盖率报告

4. **代码质量检查**
   - 文件数量统计
   - 代码行数统计
   - 测试比例检查
